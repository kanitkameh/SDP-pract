### Зад.6

Винаги трябва да има скоби за всеки оператор за да няма двусмислие.
```
Примерен вход:
(true & (true v false))
Желано дърво:
     &
    / \
true   v
      / \
  true   false
```
Бонус:
Направете програмата да оценява израза.
